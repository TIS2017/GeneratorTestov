# Generátor testov
Webová aplikácia pre doc. Šikudovú.

## Inštalácia

```
composer install
npm install
```

## Vytvorenie databázy
```
php artisan migrate
```

## Naplnenie databázy náhodnými dátami
```
php artisan db:seed
```
## Compiling assets
Run this command
```
npm run dev
```
If you want to minify output run this
```
npm run production
```
See [this](https://laravel.com/docs/5.5/mix) for more info.
